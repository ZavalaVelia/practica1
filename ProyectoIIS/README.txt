# Proyecto Final IIS – Editor de Texto y Generador de Documentos

Programa en C que permite:
- Crear y modificar texto (malloc, realloc)
- Registrar personas dinámicamente (calloc + malloc)
- Combinar texto + datos de personas
- Mostrar documentos personalizados
- Generar archivos individuales y uno general
- Liberar toda la memoria (free)
- Menú controlado con switch

## Compilación
gcc proyectof.c -o proyecto
