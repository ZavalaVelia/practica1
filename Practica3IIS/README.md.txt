# Práctica 3: Editor de Texto Básico con Memoria Dinámica

## Descripción

Esta práctica consiste en desarrollar un **editor de texto básico** que permite:

- Ingresar un texto de longitud indefinida.  
- Buscar una palabra específica dentro del texto.  
- Reemplazar todas las ocurrencias de esa palabra por otra.  
- Manejar memoria dinámica utilizando `malloc`, `realloc` y `free`.  

El objetivo es aplicar conocimientos de **arreglos**, **apuntadores**, **funciones** y **memoria dinámica** en C.
