# Práctica: Manejo de Memoria Dinámica en C

Programa en C que demuestra el uso de `malloc`, `calloc`, `realloc` y `free` mediante un sistema básico de registro y gestión de elementos almacenados dinámicamente.

## Contenido
- Inserción de elementos con memoria dinámica  
- Expansión del arreglo con `realloc`  
- Inicialización con `calloc`  
- Liberación total de memoria al finalizar  

## Requisitos
- GCC o cualquier compilador compatible con C
- Sistema operativo Windows, Linux o macOS

## Compilación
```bash
gcc main.c -o practica_memoria
