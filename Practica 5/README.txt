# Práctica 5 – Editor de Texto en C

Editor de texto básico en lenguaje C que utiliza memoria dinámica (`malloc`, `calloc`, `realloc`, `free`), arreglos tipo buffer, apuntadores y `struct` para generar documentos personalizados dirigidos a diferentes personas.

## Funcionalidades
- Crear texto
- Modificar texto
- Registrar personas (struct)
- Generar documentos personalizados (texto + persona)
- Menú con `switch`
- Manejo completo de memoria dinámica
